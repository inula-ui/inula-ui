# en-US

Use the `labelPlacement` prop to adjust the label placement.

# zh-CN

使用 `labelPlacement` 属性调整标签位置。

```tsx
import { Switch } from '@inula-ui/components';

export default function Demo() {
  return <Switch labelPlacement="left">Switch</Switch>;
}
```

# en-US

If the `expandOne` prop is `true`, the menu at the same level can expand at most one submenu.

# zh-CN

`expandOne` 属性为 `true` 时，同一层级的菜单至多展开一个子菜单。

```tsx
import { Menu } from '@inula-ui/components';

export default function Demo() {
  return (
    <Menu
      className="menu"
      width={300}
      list={[
        {
          id: 'Sub1',
          title: 'Sub 1',
          type: 'sub',
          children: [
            {
              id: 'Sub11',
              title: 'Sub 11',
              type: 'sub',
            },
            {
              id: 'Sub12',
              title: 'Sub 12',
              type: 'sub',
            },
            {
              id: 'Sub13',
              title: 'Sub 13',
              type: 'sub',
            },
          ],
        },
        {
          id: 'Sub2',
          title: 'Sub 2',
          type: 'sub',
          children: [
            {
              id: 'Sub21',
              title: 'Sub 21',
              type: 'sub',
            },
            {
              id: 'Sub22',
              title: 'Sub 22',
              type: 'sub',
            },
            {
              id: 'Sub23',
              title: 'Sub 23',
              type: 'sub',
            },
          ],
        },
      ]}
      defaultExpands={['Sub1']}
      expandOne
    />
  );
}
```
